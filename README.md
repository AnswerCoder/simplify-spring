#simplify-spring
>手写一个简化的spring
### 说明
以 Spring 源码学习为目的，通过手写一个简化版的Spring框架，了解 Spring 核心原理。

IOC:
- [x] 简单Bean容器
- [x] Bean的定义、注册、获取
- [x] 基于Cglib实现含构造函数的类实例化策略
- [x] 为bean对象注入属性和其他依赖对象
- [x] 资源加载器解析文件,注册对象
  - 从spring.xml解析和注册Bean对象
- [x] 实现应用上下文
